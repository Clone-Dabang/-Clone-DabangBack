<br>

- Java 8
- SpringBoot 2.5.2
- Gradle 7.0.2
- Spring data JPA
- MySQL 8.0

🎬`Front-end`
-  React ([React Repository이동]([https://github.com/Clone-Dabang/-Clone-DabangFront/tree/master/dabang_clone](https://github.com/Clone-Dabang/-Clone-DabangFront/tree/master/dabang_clone)))
